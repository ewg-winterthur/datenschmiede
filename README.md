# Hello World Streamlit App

Eine einfache "Hello World" Streamlit-Anwendung.

## Lokales Testen

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment auf Streamlit Cloud

1. Erstelle ein GitHub Repository
2. Lade diese Dateien hoch:
   - `app.py`
   - `requirements.txt`
3. Gehe zu [share.streamlit.io](https://share.streamlit.io)
4. Melde dich mit deinem GitHub Account an
5. Klicke auf "New app"
6. WÃ¤hle dein Repository, Branch (main) und die Datei `app.py`
7. Klicke auf "Deploy"

Die App wird in wenigen Minuten live sein!
